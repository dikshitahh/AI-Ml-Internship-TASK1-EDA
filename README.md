# AI-Ml-Internship-TASK1-EDA
Using Python Libraries like numpy, pandas on Netflix Dataset performed EDA

##Overview
This project focuses on Data Collecetion, Data Cleaning and Exploratory Data Analysis(EDA) of the Netflix dataset using Python.

#Dataset
Netflix Titles Datasets containing information about movies and tv shows available on Netflix.

##Objectives
-> Loading the dataset.
-> Identifying the missing values.
-> Handling Missing values.
-> Check and Remove duplicate records.
-> Generating summary statistics.
-> Create Visualisations.
Extracting Meaningful Insights.

##Data Cleaning
-> Filled missing values in director, cast, country, date_added, rating and duration column.
-> Checked and Removed duplicate rows.
-> Saved the cleaned dataset dor the further ananlysis.

##Exploratory data Analysis
-> Dataset overview.
-> Missing Value Analysis.
-> Duplicate record analysis.
-> Summary statistics.
-> Visualisation of data trends.

##Visualisations
-> Histogram of Movies vs tv shows
-> Line Chart for showing content release trends.
-> Scatter plot for exploratory analysis.

##Key Insights
-> Netflix contains movies than TV shows.
-> Content production increased significantly after 2015.
-> Missing values were successfully handled.
-> The dataset contains content from various countries and categories.
